🔐 VidSpark OAuth Setup

Configuration automatique de Google OAuth

📋 Instructions

1️⃣ Ouvre la Console
Clique le bouton à droite "Ouvrir Google Cloud Console"
2️⃣ Clique sur le Client OAuth
Dans la page "Credentials", cherche et clique sur "VidSpark AI" ou le client avec l'ID 665845815325-...
3️⃣ Ajoute les Origins JavaScript
Dans la section "Authorized JavaScript origins", clique "+ Add URI" et ajoute:

Origins JavaScript:

https://vidsparkpro.com
https://www.vidsparkpro.com
4️⃣ Ajoute les Redirect URIs
Dans la section "Authorized redirect URIs", clique "+ Add URI" et ajoute:

Redirect URIs:

https://vidsparkpro.com/login.html
https://vidsparkpro.com
https://www.vidsparkpro.com/login.html
https://www.vidsparkpro.com
5️⃣ Clique SAVE
Clique le bouton "SAVE" en bas de la page

⏱️ Timer & Actions

Attends après SAVE:

00:00

(Minimum 5 minutes pour que Google applique les changements)

✅ Après avoir cliqué SAVE:

  1. Clique "Démarrer Timer"
  2. Attends 5+ minutes ⏱️
  3. Clique "Tester" quand le timer est écoulé

ℹ️ Conseil: La page de test (google-oauth-test.html) va te montrer exactement ce qui marche et ce qui ne marche pas. Très utile pour déboguer!
