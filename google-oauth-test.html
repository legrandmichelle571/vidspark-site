ℹ️ À Quoi Sert Cette Page?

Cette page teste si Google OAuth est correctement configuré. Elle:

  • Charge la librairie Google Identity Services
  • Vérifie le Client ID
  • Affiche tous les logs de debug
  • Te permet de tester la connexion Google
Se connecter avec Google

❓ Étapes de Configuration Google Cloud

Si tu vois une erreur "invalid_client", il faut configurer Google Cloud Console:

  1. Va à console.cloud.google.com/credentials
  2. Clique sur ton client OAuth "VidSpark AI"
  3. Ajoute ces URLs aux "Authorized JavaScript origins":
    https://vidsparkpro.com
    https://www.vidsparkpro.com
  4. Ajoute ces URLs aux "Authorized redirect URIs":
    https://vidsparkpro.com/login.html
    https://vidsparkpro.com
    https://www.vidsparkpro.com/login.html
    https://www.vidsparkpro.com
  5. Clique sur SAVE
  6. Attends 5-10 minutes
  7. Reviens tester ici
