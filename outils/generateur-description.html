Questions fréquentes

Pourquoi la description est-elle importante ?

YouTube indexe le texte de ta description : une description riche en mots-clés naturels aide l'algorithme à comprendre ta vidéo et à la proposer dans les recherches. Les 2 premières lignes sont visibles avant le clic « plus ».

Combien de hashtags utiliser ?

3 à 5 hashtags pertinents suffisent. Les 3 premiers apparaissent au-dessus du titre de la vidéo.

Quelle longueur idéale ?

Au moins 300 caractères. Une bonne description contient : un résumé accrocheur, les mots-clés naturels, un appel à l'abonnement et tes liens.
